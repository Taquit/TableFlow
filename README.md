# Product: TableFlow
## Product Overview

This is **Project_1**, part of the P_Party portfolio project.

This project is a web application that helps manage table seating for various events. It primarily keeps track of seating arrangements, attendee payments, and payment amounts

## Key Goals

- Provide a clear and usable interface.
- To organize and track table assignments clearly, ensuring event planners know exactly who is sitting where and with whom.
- To monitor the financial status of each guest, recording whether they have paid and the exact amount contributed.
- To support the management of multiple distinct events across different physical locations within a single, centralized platform.
- To reduce the manual effort, errors, and confusion typically involved in coordinating event logistics, guest lists, and finances.
